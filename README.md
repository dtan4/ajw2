# Ajw2

One-Page Modern Web Application Generator using Client-Side Centric Model

## Installation

Add this line to your application's Gemfile:

    gem 'ajw2'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ajw2

## Usage
```
$ ajw2 MODEL_DEFINITION_FILE OUTPUT_DIR
```
